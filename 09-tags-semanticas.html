tags semanticas

as tags semanticas sao elementos HTML que descrevem claramente o significado do conteudo que elas envolvem. elas ajudam a melhorar a acessibilidade, a SEO e a estrutura do documento. aqui estao algumas das principais tags semanticas

  • <header<:define o cabeçalho de um documento ou seção.
  • <nav<:defina um conjunto de links de navegção.
  • <main<: define o conteudo principal do documento.
  • <section<: define uma seção generica de documento.
  • <article<: define um conteudo indepentende e auto-contido, como artigo de blog ou noticia.
  • <aside<: define um conteudo relacionado, mas separado do conteudo principal, como uma barra lateral.
  • <footer<: define o rodapé de um documento ou seção

Usar essas tags semanticas ajuda os motores de busca a entender melhor o conteudo do seu site e melhora a experiencia do usuario, para aqueles que utilizam leitores de tela.
